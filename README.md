# First Project to push on GitHub

## Installation:

### 1. Close repo
`git clone https://github.com/krutskikh/HelloWorld.git`

### 2. Go to project
`cd HelloWorld`

### 3. Install dependencies
`composer install`

## Running tests

### Use `phpunit {test name}`
### Example: `phpunit src/Tests/AssertTrueFalse.php`
